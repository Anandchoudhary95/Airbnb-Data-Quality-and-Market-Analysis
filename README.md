# Project Title: Airbnb Data Quality and Market Analysis

## 1. Introduction

This project explores Airbnb listing data obtained from the Inside Airbnb platform.
The work focuses on two important areas:

**Data Quality Assessment** – verifying whether the dataset is reliable for analysis.

**Market Analysis** – extracting insights that help understand pricing, availability, demand, and supply across neighbourhoods and room types.

## 2. Data Source

Dataset used: https://insideairbnb.com/get-the-data/

**Main columns:**
- id
- name
- host_id
- host_name
- neighbourhood
- room_type
- price
- minimum_nights
- number_of_reviews
- last_review
- reviews_per_month
- calculated_host_listings_count
- availability_365
- number_of_reviews_ltm
- license
